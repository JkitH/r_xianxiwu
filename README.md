# 饕食屋 - 一頁式資訊網頁

## 📋 專案概述

本專案為新北市永和區著名平價日式豬排飯餐廳「饕食屋」建立的一頁式靜態資訊網頁。網頁整合了來自 Google Maps、Uber Eats、Facebook 等多個平台的真實資訊和評論，並進行了全面的 SEO 優化。

## 🚀 使用方式

### 本地預覽
1. 解壓 `restaurant_website.zip` 檔案
2. 在瀏覽器中開啟 `index.html`
3. 或使用本地伺服器：
   ```bash
   python3 -m http.server 8000
   ```
   然後訪問 `http://localhost:8000`

### 部署到網路
1. 將整個 `restaurant_website` 資料夾上傳到網頁伺服器
2. 確保 `index.html` 在根目錄
3. 確保 `images` 資料夾與 `index.html` 在同一目錄
